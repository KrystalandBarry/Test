1.进入目录命令
	使用相对路径进入某个目录 cd ./
	使用绝对路径进入某个目录 cd /root/..
	回家目录
2.列出资源命令
	单独使用ls命令查看当前目录下非隐藏的所有目录和文件 ls  
	查看当前目录下隐藏的目录和文件 ls -a
	查看当前目录下隐藏的目录和文件，但是不显示“.”和“..” ls -A
	列出当前目录下资源的详细信息 ls -l
	列出当前目录下资源的详细信息，使用快捷方式 ll
	递归查看某个目录下的资源 ls -R
	在列出资源时给目录后面附加斜杠  ls -F
	通过参数组合在列出详细信息的同时给目录后附加斜杠 ls -lF
	使用命令管道将当前目录下所有文件筛选出来 ls -l |grep ^-
	使用命令管道将当前目录下所有目录筛选出来 ls -l | grep ^d
3.创建目录命令
	在当前目录下创建一个新的目录 mkdir
	在指定目录下创建一个新的目录 mkdir /root/..
	一次性创建多层目录 mkdir -p ./test1/test2/
4.创建一个空的文件 touch ..
5.复制
	将一个文件复制到另外一个目录下 cp .. ./abc/
	将一个文件复制到另外一个目录下，并修改文件名 cp .. ./abc/ad
	复制目录 cp -r 
6.删除
	常规删除一个文件 rm
	强制删除一个文件 rn -f
	常规删除一个目录 rm -r
	强制删除一个目录 rm -rf ..
7.移动
	移动一个文件 mv
	移动一个目录
	重命名一个文件
8.VIM编辑器
	浏览文件内容
		回到第一行开头 gg
		到最后一行开头 shift + G
		前往指定行 n + G
		上下左右移动光标 hjkl
	编辑文件内容
		删除当前行 dd
		删除当前行开始的多行 dnG
		复制当前行 yy
		复制当前行开始的多行 yny
		粘贴 p
		撤销 u
		重做 ctrl r
		修改一个字符 r
	进入编辑模式
		在当前光标前，以插入模式进入 i
		在当前行开始位置，以插入模式进入 I
		在当前光标后，以插入模式进入 a
		在当前行末尾，以插入模式进入 A
		从当前光标的下一行以插入模式进入 o
		从当前光标的上一行以插入模式进入 O
	常用指令
		保存  :w
		保存退出 :wq
		强制保存 :w!
		退出  :q
		强制退出 :q!
		强制保存退出 :wq!
		另存为 :w ....
		查找关键词  /
		替换关键词  %s/keyword/g
		显示行号 set nu
	区块模式
